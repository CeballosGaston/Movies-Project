# 🎬 Movies — Cinematic Preferences

The **Movies** project is a web application for managing cinematic preferences. It allows users to query real-time movie information and save favorites with custom ratings.


## 📦 Installation & Setup

###  Clone the repository

```bash
git clone (https://github.com/CeballosGaston/Movies-Project.git)
cd movies-app

---

## 🎯 Platform Objective

Develop a dynamic and personal web platform where any user can:

- Explore current or classic movies.
- Discover detailed information about each movie.
- Add movies to a favorites list.
- Rate movies and generate a personal ranking.

---

## 🌐 Expected Service

A private-access web application that provides:

- Movie search and browsing.
- Complete and up-to-date movie details.
- Favorites system.
- Personal rating system.
- Personal movie ranking.
- User authentication and registration system.
- Personalized data persistence.


## 🎯 Target Audience

The project is aimed at **movie enthusiasts and cinema lovers** who want to manage their own movie database.

### Interests

- Discover movies across different genres, eras, and countries.
- Access technical and artistic information (actors, directors, screenplay, music, etc.).

### Needs

- Create personalized lists (favorites).
- Search by title, actor, director, genre, or keyword.
- Quickly access detailed movie pages.
- Save personal preferences.

---

## ⚙️ Functional Requirements

### 🎞️ Movie Exploration

- Welcome screen with direct access to exploration.
- Real-time movie listing.
- Basic information: title, poster, year, and rating.
- Filters and search by genre, rating, or trends.

### 🎬 Movie Details Page

- Synopsis.
- Cast.
- Director.
- Release date.
- Trailer and extended information. (WIP)

### 👤 Actor Details Page

- Photo and personal information.
- Nationality.
- List of movies they appear in.

### 🎥 Director Details Page

- Photo and personal information.
- Nationality.
- Full filmography.

### 🔐 User Management

- Registration and login with email and password.
- Restricted access to authenticated users.
- Optional integration with external services (Google, Facebook, etc.).

### ❤️ Favorites and Personal Ranking

- Add movies to favorites.
- Personal rating (1–10).
- Personal favorites list.
- Edit or remove ratings and favorites.

### 🧭 Navigation & Interface

- Main menu with:
  - Home
  - Explore
  - Favorites
  - Profile



## 🧱 Technical Requirements

- Use of an external movie API (TMDB recommended).
- Own database for users, favorites, and ratings.
- SPA (Single Page Application).
- Main framework: React.
- Mobile-first design.
- Lightweight backend for data persistence.
- Unique URLs for movies, actors, and directors.
- Deployment on a server or cloud platform.
